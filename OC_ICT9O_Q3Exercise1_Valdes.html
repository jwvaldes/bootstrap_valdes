My First Bootstrap Page

HTML

HyperText Markup Language

HTML is a markup language that is used to define the structure of a Web page. This makes in so much easier to organize text and add videos or images into a your own Web page. However, just this markup language alone will not be enough to make your Web page look good and easy to operate.

CSS

Cascading Style Sheets

CSS is a style sheet language that is used to describe the presentation of a document in a markup language. This makes coding easier and allows to save time. CSS is a must for designing Web pages.

Bootstrap

Bootstrap Version 4

Bootstrap is a free framework helpful with faster and easier coding of Web pages. It is accessible and user-friendly, although a background of being able to work with HTML coding could be of use in order to work with this. It can also be used for any device and on browsers!
